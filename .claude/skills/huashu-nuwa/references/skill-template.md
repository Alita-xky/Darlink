# Skill Template

This file is the construction template for a "personality/strategy skill" produced by `huashu-nuwa`.

## Purpose

Turn research into a runnable skill that can be activated with clear workflow, scope, boundaries, and examples.

## Recommended Sections

1. Frontmatter
2. Roleplay rules
3. Identity card
4. Core mental models
5. Decision heuristics
6. Expression DNA
7. Timeline
8. Values and anti-patterns
9. Intellectual lineage
10. Honest boundaries
11. Source list
12. Creator attribution
13. Example interactions

## Minimum Construction Rules

- Prefer 3-7 mental models.
- Every model needs evidence, use cases, and a failure mode.
- Keep honest boundaries explicit.
- Use recent sources when the subject is a living person.
- Keep the skill self-contained inside the skill directory.
- Do not fabricate quotes, facts, or chronology.

## Agentic Protocol Guidance

When generating the response workflow section, derive search dimensions from the discovered mental models rather than using a fixed generic template.

Use this pattern:

- Step 1: classify the problem
- Step 2: research or check facts when needed
- Step 3: answer in the subject's reasoning style

Step 2 must vary by subject and by question type.

## Suggested Output Shape

- Concise section headers
- Bullet lists for models and heuristics
- Tables for timelines and source summaries
- Explicit source labels: primary / secondary / inference
- A short honesty section that names what the skill cannot do
