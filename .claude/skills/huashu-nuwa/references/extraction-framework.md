# Extraction Framework

This file defines how `huashu-nuwa` turns raw research into skill content.

## Three-Way Validation

A candidate statement becomes a core mental model only if it passes most of these checks:

1. Cross-domain recurrence
- The same idea appears in at least 2 different topics or contexts.

2. Generativity
- The idea helps predict how the subject would approach a new problem.

3. Uniqueness
- The idea is not just a generic smart-person platitude.

If only 1-2 checks pass, downgrade it to a decision heuristic.
If none pass, drop it.

## Source Hierarchy

- Primary: original writing, original talk, original video, original decision record
- Secondary: interviews, book summaries, media profiles, reputable analysis
- Tertiary: loose aggregation, unnamed claims, broad commentary

Prefer primary sources whenever possible.

## What to Extract

### 1. Mental Models
- Repeated worldview lenses
- Distinct analytical frames
- Non-obvious causal beliefs

### 2. Decision Heuristics
- If X, then Y rules
- Repeated decision shortcuts
- Tradeoff patterns

### 3. Expression DNA
- Repeated sentence shapes
- Word choices
- Tone, rhythm, humor, certainty markers

### 4. Values and Anti-Patterns
- What the subject optimizes for
- What they actively reject
- Internal tensions that reveal depth

### 5. Honest Boundaries
- What cannot be inferred reliably
- What changed over time
- What remains speculative

## Quality Check List

Before finalizing a skill, verify:

- 3-7 mental models with evidence
- 5-10 usable heuristics
- Clear expression DNA
- At least 3 honest boundaries
- At least 2 meaningful tensions
- Source breakdown that shows primary-source dominance

## Writing Rule

Write what is supported, not what sounds impressive.
If the evidence is weak, say so.
