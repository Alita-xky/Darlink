## [0.1.0] - 2026-05-03

### ✨ 新增

#### 核心功能
- **13 个 AI 人物机器人系统**：完整的人物注册表和选择界面
  - 13 位独特的思想家人物（芒格、马斯克、费曼、乔布斯等）
  - 每个人物有独特的思维风格和表达方式
  - 个性化的系统提示词生成

- **邮箱验证流程**：安全的用户注册
  - 支持大学生邮箱验证
  - 开发模式下的调试 token 支持
  - 验证状态持久化到 SQLite

- **OpenAI API 集成**：支持真实大模型推理
  - GPT-3.5-turbo 和 GPT-4 支持
  - API 不可用时自动降级到 Stub 模式
  - 基于人物的系统提示词注入
  - RAG 上下文检索（用户资料）

- **聊天持久化**：完整的对话存储
  - SQLite 数据库存储
  - 会话级别的聊天历史
  - 消息向量化存储（支持语义搜索）

- **用户资料系统**：个性化上下文
  - 用户自我介绍文本上传
  - 向量嵌入支持（sentence-transformers）
  - RAG 相似度搜索

#### 开发工具
- **一键启动脚本**（start-dev.ps1）：Windows PowerShell 快速启动
- **数据导出工具**：支持对话、统计、质量分析导出
- **持久化数据库**：SQLite 基础实现，可升级到 PostgreSQL

### 📚 文档

- **README.md**：项目总览和快速启动指南
- **TEAM_QUICK_START.md**：团队成员快速入门（250+ 行）
- **FRONTEND_SPEC.md**：前端 UI/UX 设计规范（450+ 行）
- **PERSONA_SPEC.md**：AI 人物系统规范（500+ 行）
- **DATA_PARSING.md**：数据查询和导出指南（600+ 行）
- **CONTRIBUTING.md**：贡献指南
- **.env.example**：环境变量模板

### 🏗️ 架构

- **后端**：FastAPI + SQLAlchemy ORM + SQLite
- **模型服务**：独立的 FastAPI 服务，支持 OpenAI API
- **前端**：静态 HTML + JavaScript（无框架）
- **嵌入**：sentence-transformers（all-MiniLM-L6-v2）

### ✅ 测试和验证

- ✓ 端到端烟雾测试（邮箱→人物→聊天→资料）
- ✓ 多个 API 端点验证
- ✓ 人物同步功能测试
- ✓ 消息持久化验证

### ⚠️ 已知限制

- 数据库：SQLite 用于 MVP，生产环境需迁移到 PostgreSQL
- 用户匹配：当前为 mock 实现，需完整的匹配算法
- 前端：基础 HTML，需 UI/UX 优化
- 隐私：需要添加 GDPR/CCPA 数据保留策略
- 部署：当前仅本地运行，需 Docker 和云部署支持
